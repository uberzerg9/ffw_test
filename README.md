### Exercise One.

## 1 step:

Need to install node-modules:

```js
    npm i --save-dev
```

## 2 step:

First webpack build:

```js
    npm run build
```

Watch mode:

```js
    npm run watch
```

## 3 step:

Browser sync (need to instal):

```js
browser-sync start --proxy 'domain-name' (open server required)
```
